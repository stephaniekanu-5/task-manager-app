# 🧩 Task Manager App

A Kanban-style task management application built with React.

---

## 🚀 Features

- Drag-and-drop task management  
- Add, edit, and delete tasks  
- Task priority and deadlines  
- LocalStorage persistence  
- Dark mode toggle  

---

## 🛠️ Technologies

- React  
- JavaScript (ES6+)  
- CSS  
- @hello-pangea/dnd  

---

npm install
npm run dev
